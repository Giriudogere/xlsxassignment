Upload .xlsx file to backend

A simple application to upload the values of .xlsx file to the backend database.

  • Click on choose file to select file
  • By default it will show only .xlsx files present in the folder
  • Once file is selected, click on upload button to upload the data to the backend db
  • Wait for the confirmation message once the upload is completed
